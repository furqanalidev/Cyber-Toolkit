Cybersecurity Toolkit
====================

A modular toolkit for cybersecurity tasks including port scanning, vulnerability detection, packet sniffing, encryption, web utilities, and ML-based anomaly detection.

Usage:
------
Run the toolkit:
    python toolkit.py

Features:
---------
- Port Scanner
- Vulnerability Scanner
- Packet Sniffer
- Encryption/Decryption
- Web Tools
- ML Security

Each module can be extended for more features.
